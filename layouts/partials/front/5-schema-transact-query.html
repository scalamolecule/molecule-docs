Define schema

Define and compile schema (seldom) 
trait Person {
  val name    = oneString
  val age     = oneInt
  val address = one[Address]
}
trait Address {
  val street = oneString
}

> cd yourProjectRoot
> sbt compile

Transact data

  List(
    ("Lisa", 20, "Broadway"),
    ("John", 22, "Fifth Avenue")
  )
